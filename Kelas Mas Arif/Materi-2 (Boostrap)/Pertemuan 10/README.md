### colom pada boostrap
- jika ingin untuk membuat colom ukuran dekstop
    `lg : untuk dekstop`

- jika ingin untuk colom ukuran medium
    `md : untuk medium`

- jika untuk colom ukuran small
    `sm : untuk kecil`

- jika untuk colom yang sangat kecil 
    `xs : untuk yang sangat kecil`

Cara penulisan : 

```html 

< div class='col-pilihannya(md)-jumlah kolomna ( maksuimal : 12) = ( 1 satu kolom ' >

```

- jika kita hanya menggunakan `col` maka akan membuat colom sesuai dengan content yang di isi

    ex: 
    ```html
    <div class='col'>
            Content
    </div>
    <div class='col'>
        content
    </div>
    ```
